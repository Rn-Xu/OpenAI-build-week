## Title

Get up! — OpenAI Build Week Demo | Codex + GPT-5.6

## Description

Get up! is an action coach designed to help people overcome the friction of starting.

Instead of generating a long and overwhelming task list, Get up! turns a goal into one small, practical action at a time. After the user clicks Done, it provides the next manageable action. If the step still feels difficult, Make it smaller creates an easier recovery action.

The public deployment runs in a transparent, credential-free demo mode. The repository also includes an optional OpenAI Responses API path using GPT-5.6 for personalized guidance when an API key is configured.

Codex with GPT-5.6 was the primary development workflow for product design, structured prompt design, Streamlit implementation, output hardening, automated interaction testing, deployment, and submission preparation.
