Get up! — OpenAI Build Week Demo

## Description

Get up! is an AI action coach designed to help people overcome procrastination and take the first step.

Instead of generating long and overwhelming task lists, Get up! turns a user's goal into one small, practical action at a time. After the user responds, it provides the next manageable action or makes the step smaller when the user feels stuck.

Built during OpenAI Build Week with Codex, the OpenAI API, Python, and Streamlit.
