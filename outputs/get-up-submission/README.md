# Get up! — OpenAI Build Week submission pack

This folder contains the copy and media plan for the public project page.

## Recommended submission order

1. Paste `project-story.md` into **Project Story**.
2. Add the tags and links from `submission-fields.md`.
3. Upload the five images from `media/` in the order listed in `media-plan.md`.
4. Record the demo with `video-script.md`.
5. Upload the video as **Unlisted** on YouTube and use `youtube-description.md`.
6. Complete `final-checklist.md` before submitting.

## Important assumptions

This pack assumes the prototype can:

- accept a user's goal in natural language;
- return one small, concrete first action;
- provide the next step after completion;
- make a step smaller when the user is stuck;
- display a simple completion state.

It also assumes the stack is **OpenAI API, Codex, Python, Streamlit, and GitHub**. Remove anything that is not true before publishing. Replace all `[ADD ...]` placeholders, and replace UI concept images with real screenshots when the working app is available.
